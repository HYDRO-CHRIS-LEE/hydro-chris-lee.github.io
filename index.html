About Me Projects

My Story

I am Jonghyeok Lee, a dedicated hydrologist with a strong background in Civil and Environmental Engineering and Economics. My research bridges the gap between hydrological modeling and economic analysis, focusing on how human activities impact water resources. By developing innovative solutions such as enhancing global water resource assessments with CLM5 and improving streamflow predictions using Graph Neural Networks, I aim to build comprehensive, explainable models that address both environmental and economic challenges.

Land-Surface Model

I work on the Community Land Model(CLM5), with a particular focus on hydrology and water resource processes.

Machine Learning

I apply machine learning techniques, particularly graph neural networks, to improve streamflow prediction by incorporating hydrological insights.

Programming

I utilize a variety of programming languages and tools for scientific computing, model development, and data analysis, using a Linux server environment.

Featured Projects

Incorporating Sectoral Groundwater Withdrawals in CLM5

This project focuses on integrating sectoral groundwater withdrawals in the CLM5 land surface model. It includes building the model's framework and selecting the parameters, along with analyzing global sectoral water withdrawal results.

Check it out!

Improving Streamflow Prediction with Graph Neural Network

In this project, we applied different graph neural networks (T-GCN, ASTGCN, Graph WaveNet) to improve streamflow prediction. The results demonstrate the changing performance of the GNN models as we move downstream.

Check it out!

Automating Hydrologically-Informed Adjacency Matrices in ArcGIS

This project involves automating the creation of hydrologically-informed adjacency matrices using ArcGIS. The video shows the execution of the code that automates this process, making it more efficient for hydrological studies.

Check it out!

Measuring the Effects of Changes in Real GDP Caused by Weather Shocks

Using local projection methods, this project measures the impact of a 10% increase in weather shocks on real GDP. The impulse responses with 68% and 95% confidence intervals are shown for various countries and weather shocks, such as temperature and precipitation shocks in South Korea and France.
